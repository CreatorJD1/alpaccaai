# Alpecca VRoid Accessory Workbench

Experimental blue hair-clip work now lives under:

`data/alpecca_art_source/vrm_experiments/accessory_workbench/`

This workbench keeps the current VRoid Studio state untouched and provides an
open local proxy for the locked small glossy blue X/bone-bow clip on Alpecca's
left side. Use the OBJ/MTL and `accessory_spec.json` for VCS, Blender, Unity, or
Three.js attachment tests after VRM export.

VRoid custom accessory creation is still a manual/import route: use
`Accessories > Import as Custom Item` when a matching `.vroidcustomitem` is
available. Do not route the clip through skin, hoodie textures, animal ears,
hats, collars, or unrelated presets.
