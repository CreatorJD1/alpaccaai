## Decision

Continue v11 as the main experimental Alpecca VRoid branch.

## Why v11 Stays Active

- v11 has the stronger Alpecca identity base:
  - white/lavender hair direction,
  - stronger blue eyes,
  - hoodie silhouette,
  - imported lanyard/ID layer,
  - single ahoge continuity.
- v12 preserves the user-adjusted v0 state but currently reads as a simpler base:
  - blonde/yellow hair,
  - black tank top,
  - no hoodie-jacket,
  - no lanyard/ID,
  - no visible outfit identity markers.
- v12 should remain available as a body/face fallback reference, but promoting it
  would require reapplying most of the v8-v11 identity work.

## Next v11 Repair Targets

1. Shrink and reposition the imported lanyard/ID layer so it sits naturally on
   the chest instead of dominating the hoodie front.
2. Add or import the left-side blue bone/bow hair clip.
3. Paint or import the lower lavender-blue hair gradient.
4. Move the closed hoodie proxy closer to the open warm-ivory hoodie-jacket read
   through texture cues first, then silhouette work if VRoid allows it.
5. Add the black right-leg thigh strap and cream/blue boot details.

## Rule

Do not validate v11 gates from v12 screenshots. Reopen v11 before gate updates.
