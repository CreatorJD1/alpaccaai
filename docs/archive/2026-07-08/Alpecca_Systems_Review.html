Alpecca — Systems Review

A comprehensive map of every system that powers the local companion app.

Generated 2026-06-14 · grounded in the codebase: 40 modules / ~8,100 LOC in alpecca/, 83 HTTP/WS endpoints, 13 config groups, 14 scripts · branch main @ f286f03

The spine: one loop. Everything hangs off a single readable loop, run every turn and orchestrated by mind.py — kept as plain Python rather than hidden inside a framework. 
sense  →  update mood  →  recall memory  →  generate reply  →  persist

1 · Core orchestration & brain

ComponentRole
mind.py (1,419)The Core Mind. Ties senses→mood→memory→LLM→persistence; idle self-direction, reflection, bounded tool-call chaining, animation authoring.
prompts.pyTurns live mood + memories + situation + self-report into the system prompt — where the personality lives.
config.pyAll tunable knobs across 13 classes (Emotion, Home, Files, RigData, Voice, Hearing, Vision, Proactive, Reflection, Actions, Computer, Portrait, OpenClaw).
server.pyFastAPI + WebSocket host; 83 endpoints; token-auth middleware.
app.pyNative desktop window (pywebview) + LAN/tunnel orchestration.
Tiered LLMIn mind.py: local reason (configured ALPECCA_MODEL) + fast (gemma), optional deep (Anthropic/cloud). Chat always local; only her hardest self-acts request deep. Offline stub fallback.

11 · Runtime / infrastructure

Review observations

Health. 149 / 149 core tests pass; the full loop runs offline (no Ollama- or Windows-dependent test, by design).
Architectural throughline — GROUNDING. Every self-report reads from real internals; no confabulation. This is the load-bearing design constraint, enforced across introspection, soul, journal, and selfmod.
Privacy posture is strong. Frames and audio are never stored (only short text descriptions survive); outward channels are opt-in and charter-bounded; all sense data stays local.
Biggest open edge. A single finished rigged avatar figure in the 3D home — the affect, channels, and render tiers all exist, but the end-to-end rigged render (gated on a GPU art-decomposition step) is the incomplete piece.
